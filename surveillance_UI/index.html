Departments

Unknown-case link review

Suggested links between Unknown cases seen on the same camera within a short window, whose body similarity is plausible but below the automatic bar. Each suggestion shows the score, time gap and both parties' evidence — nothing is merged without your approval, and every approval is audited and undoable.

Add employee(s)

Register one person now, or bulk-import a whole roster.

Add a single employee

Enter their name, employee id and department, then upload one or more clear, front-facing face photos. The system extracts the face and remembers this person.

Add many (XLSX / CSV / ZIP)

Upload an XLSX/CSV roster, or a ZIP with the roster plus photos named by employee id (e.g. E1042_1.jpg). Columns: external_id (employee id, required — the idempotency key, re-imports update, never duplicate), name (required), department, email. You always get a preview before anything is written.

Daily reset

Clear all Unknowns (people not yet confirmed as Visitors). Confirmed Visitors and Employees are kept, so returning people are still recognised. Runs automatically every midnight.

Danger zone

Permanently delete every person, event and snapshot from the database. Cameras are kept. This cannot be undone.
